Introduction to the Web

World Wide Web

HTML- HyperText Markup Language
HTML is the main type of document on the web
Major pieces of the web are: myself, computer, internet, and servers.
This Components of the Web video gives a perfect quick overview.
Basically, the web is a bunch of computers that communicate with each other.
When a person goes to a web page like www.udacity.com, their computer sends
a HTTP Request to a server. The server finds the appropriate HTML document
and sends it back to the user's computer where a web browser interprets
the page and displays it on the user's screen.

HTML Attributes

HTML documents are made of HTML elements. When writing in HTML,
we tell browsers the type of each element by using HTML tags.
This Intro to HTML Tags explains everything well.
HTML elements are either inline or block. Block elements form an
"invisible box" around the content inside them.

Computers are Stupid

Why are computers stupid? The reason computers are stupid is because
they interpret instructions literally and programmers need to write exactly
the way a computer understands.
